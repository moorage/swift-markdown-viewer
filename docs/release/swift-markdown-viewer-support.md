## Overview

Swift Markdown Viewer is a free app for viewing Markdown folders on macOS, iPhone, and iPad.

## Supported workflow

- Open a folder that contains `.md` files.
- Browse the folder from the sidebar.
- View local images, animated images, and local video referenced from those Markdown files when the files are inside the opened folder.

## Troubleshooting

If a document or media file does not appear:

- confirm the file lives inside the opened folder
- confirm the file extension is supported by the app
- reopen the folder from Finder or the Files picker if sandbox access has expired
